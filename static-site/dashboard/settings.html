Data Syncing

With an account, how your journal entries are stored can be adjusted here.

Offline Only is the default. This means that the journal entries are stored on the device accessing the website. This is ideal if you plan to use the website on only one device or have the website installed on a device. The Offline and Online setting means that journal entries are stored primarily on one device and backedup to the Bytesized Journal web server. This is ideal if you want to extend the capability of the website to allow another device to access the journal entries after being logged in. The online only setting is for when you only want to access journal entries stored on the Bytesized Journal web server without using any storage on the device accessing the website. Please review your selection before confirming.
